# Linuxでの環境構築

## 1. Gitのインストール

OS標準のパッケージマネージャーを使用してインストールします
- DebianベースのOSの場合(Ubuntuなど)
    ```bash
    sudo apt install git
    ```
- Red HatベースのOSの場合(Fedoraなど)
    ```bash
    sudo yum install git
    ```

    ```bash
    sudo dnf install git
    ```
