# adf-core-python

## Development Environment

### Prerequisites

- Python (3.12 or higher)
- Poetry (1.8.3 or higher)

### Installation

```bash
poetry install
```

### Run Agent

```bash
poetry run python ./adf_core_python/launcher.py

# get help
poetry run python ./adf_core_python/launcher.py -h
```

### Build

```bash
poetry build
```

### Pre Commit

```bash
poetry run task precommit
```
