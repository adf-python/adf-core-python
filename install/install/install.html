Installation

This section explains how to install the package.

Prerequisites

Before installing, please check the following prerequisites:

  • Python 3.12 or higher

  • pip

Installing the package

To install the package, run the following command:

pip install git+https://github.com/adf-python/adf-core-python.git

Verifying the installation

To verify the installation, run the following command:

pip show adf_core_python

If the package is installed correctly, the package version number, etc., will be displayed.
